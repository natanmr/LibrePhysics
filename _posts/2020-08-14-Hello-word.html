Olá, mundo! A frase e bem simbolica para o mundo da computação, afinal é o ponto de partida em qualquer uso de software e linguagem de programação. Nos como amantes de física computacional não deixamos de colocar nesse post o nosso olá, Mundo!

Esse tal de software livre

As palavras software livre e código aberto soam familiar? Para muitas pessoas, principalmente distantes da computação, não. Mas, saiba que no mundo da computação essas palavras ocupam um papel central.

Antes, precisamos entender oque são licencas de software e quais implicações para o usuário.

Licencas de software

Licença de software é um documento contratual que determina como um código ou programa é tratado, quais são as permissões de acesso ao código fonte, ao uso e à distribuição. Existem três categorias de licenças de grande importância na tecnologia, são eles:

  • Software proprietário: Normalmente, para que se possa utilizar, copiar, ter acesso ao código-fonte ou redistribuir, deve-se solicitar permissão ao proprietário, ou pagar para poder fazê-lo. Será necessário, portanto, adquirir uma licença.
  • Software livre: É uma licença permissiva que fornece acesso sem restrição ao código fonte e permissão de execução do Software. As regras de uso e divulgação variam de acordo com o que o responsável pela distribuição estipula.
  • Software Open source: Nesse nessa licença há grande permissividade ao acesso ao código fonte de um software, apesar de não necessariamente ser livre a sua distribuição por terceiros.
